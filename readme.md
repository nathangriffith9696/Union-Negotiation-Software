# Union Negotiation Software

A web platform for managing union bargaining workflows, including proposals, bargaining sessions, districts, locals, and notes.

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- React

## Current Status
Initial scaffold with local development running.

## Planned Features
- Proposal tracking
- Bargaining session timeline
- District and local management
- Proposal comparison
- Notes and negotiation history
